Breve descrizione della serie scelta

ID: GSE17913

La serie è parte di uno studio che ha analizzato proprio l’effetto del fumo di sigaretta sul trascrittoma della mucosa umana. Lo studio ha trovato che 32 geni hanno mostrato un’espressione aumentata e 9 geni hanno mostrato un’espressione ridotta nella mucosa orale dei fumatori rispetto ai non fumatori.

Il campione originariamente era n=80, ma ne sono stati depositati 79 perché un campione è stato escluso in base a una misura di qualità.

Ho usato il codice generato da GEO2R e plotly per la parte del volcano plot

Codice 
library(GEOquery)
library(limma)
library(umap)
library(plotly)

gset <- getGEO("GSE17913", GSEMatrix =TRUE, AnnotGPL=TRUE)
if (length(gset) > 1) idx <- grep("GPL570", attr(gset, "names")) else idx <- 1
gset <- gset[[idx]]

# make proper column names to match toptable 
fvarLabels(gset) <- make.names(fvarLabels(gset))

# group membership for all samples
gsms <- paste0("11101101001010001001111111100111011010111000100001",
        "00010010010010110101010101001")
sml <- strsplit(gsms, split="")[[1]]

# log2 transformation
ex <- exprs(gset)
qx <- as.numeric(quantile(ex, c(0., 0.25, 0.5, 0.75, 0.99, 1.0), na.rm=T))
LogC <- (qx[5] > 100) ||
          (qx[6]-qx[1] > 50 && qx[2] > 0)
if (LogC) { ex[which(ex <= 0)] <- NaN
  exprs(gset) <- log2(ex) }

# assign samples to groups and set up design matrix
gs <- factor(sml)
groups <- make.names(c("smoker","nonsmoker"))
levels(gs) <- groups
gset$group <- gs
design <- model.matrix(~group + 0, gset)
colnames(design) <- levels(gs)

gset <- gset[complete.cases(exprs(gset)), ] # skip missing values

fit <- lmFit(gset, design)  # fit linear model

# set up contrasts of interest and recalculate model coefficients
cts <- paste(groups[1], groups[2], sep="-")
cont.matrix <- makeContrasts(contrasts=cts, levels=design)
fit2 <- contrasts.fit(fit, cont.matrix)

# compute statistics and table of top significant genes
fit2 <- eBayes(fit2, 0.01)
tT <- topTable(fit2, adjust = "fdr", sort.by = "B", number = Inf)

tT <- subset(tT, select = c("Gene.symbol", "Gene.title", "logFC", "AveExpr", "t", "P.Value", "adj.P.Val", "B"))

tT <- tT[tT$Gene.symbol != "", ]

Dato che i geni erano troppi ho implementato questa logica: tengo i 100 più differenzialmente espressi (B-statistics) e prendo l’1% dei rimanenti per vedere comunque i punti alla base del grafico.

# Ordina per B-statistic (decrescente)
tT_ordered <- tT[order(tT$B, decreasing = TRUE), ]

# Mantieni i top 100 geni per B-statistic
top_genes <- tT_ordered[1:min(100, nrow(tT_ordered)), ]

# Geni rimanenti
remaining_genes <- tT_ordered[(min(100, nrow(tT_ordered)) + 1):nrow(tT_ordered), ]

# Campiona il 25% dei geni rimanenti
set.seed(42)  # per riproducibilità
sample_size <- floor(0.01 * nrow(remaining_genes))
sampled_genes <- remaining_genes[sample(nrow(remaining_genes), sample_size), ]

# Combina i due set
tT_filtered <- rbind(top_genes, sampled_genes)

# Mantiene solo il primo nome e la prima descrizione se sono presenti più geni separati da ///
tT_filtered$Gene.symbol.short <- sapply(strsplit(as.character(tT_filtered$Gene.symbol), "///"), `[`, 1)

tT_filtered$Gene.title.short <- sapply(strsplit(as.character(tT_filtered$Gene.title), "///"), `[`, 1)

Spiegazione della B-statistics

La B-statistics in sostanza è un parametro che combina LogFC e P value.

fit2 <- eBayes(fit2, 0.01)

eBayes è una funzione del pacchetto limma che applica un approccio bayesiano empirico per stabilizzare le stime della varianza tra geni e migliorare l’affidabilità statistica quando il numero di campioni è piccolo (n<100 in genere).

Con 0.01 sto dicendo che mi aspetto che l’1% dei geni siano differenzialmente espressi; la funzione usa questa mia aspettativa a priori per capire se un punto è rilevante o meno rispetto al rumore di fondo e in questo modo calcola il valore B.
